Build Sri Lanka's most modern recruitment platform focused on:

# 2. Market Problems

### Current Problems in Sri Lanka

### Existing Platforms

* [TopJobs.lk](https://topjobs.lk?utm_source=chatgpt.com)
* [XpressJobs](https://www.xpress.jobs?utm_source=chatgpt.com)
* [JobEka.lk](https://jobeka.lk?utm_source=chatgpt.com)

### Major Weaknesses

| Problem                 | Impact                                     |
| ----------------------- | ------------------------------------------ |
| Old UI/UX               | Poor user experience                       |
| Weak filtering          | Users struggle to find relevant jobs       |
| No AI matching          | Manual searching                           |
| Fake employers          | Trust issues                               |
| Poor mobile experience  | Candidate drop-off                         |
| No application tracking | Lack of transparency                       |
| No salary insights      | Difficult decisions                        |
| No skill assessment     | Employers receive low-quality applications |
